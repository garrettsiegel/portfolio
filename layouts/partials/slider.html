Shohei Ohtani Value Explorer

Developed for ESPN's visual storytelling team, this dynamic web experience delves into the multi-faceted value of baseball sensation Shohei Ohtani. Utilizing HTML, CSS, JS, and Hugo, I engineered a "ticker" that updates in real-time as users scroll, thanks to the integration of GSAP ScrollTrigger and matchMedia. The project garnered significant user engagement and reached the front page of espn.com.
